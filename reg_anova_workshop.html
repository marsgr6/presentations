Resumen de Regresin Lineal

La regresin lineal es una tcnica estadstica que modela la relacin entre una variable dependiente ($y$) y una o ms variables independientes ($x$), utilizando una ecuacin lineal:

$$ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_k x_k + \epsilon $$

Donde:

  • $\beta_0$ es el intercepto o trmino independiente.
  • $\beta_1, \beta_2, \dots, \beta_k$ son los coeficientes asociados a las variables independientes.
  • $\epsilon$ es el trmino de error, que representa la variabilidad no explicada.

Anlisis de los Coeficientes

Los coeficientes $\beta_i$ indican la magnitud y la direccin del efecto de cada variable independiente sobre la variable dependiente:

  1. Magnitud:
    • Un coeficiente grande implica un cambio significativo en $y$ por cada unidad de cambio en $x_i$, manteniendo las dems variables constantes.
  2. Signo:
    • $\beta_i > 0$: Relacin positiva (al aumentar $x_i$, $y$ tambin aumenta).
    • $\beta_i < 0$: Relacin negativa (al aumentar $x_i$, $y$ disminuye).
  3. Significancia:
    • Se evala mediante pruebas de hiptesis ($H_0: \beta_i = 0$) usando el valor $p$. Si $p < 0.05$, el coeficiente se considera significativo.
  4. Interpretacin en regresin simple:
    • $\beta_1$: Representa el cambio promedio en $y$ por cada unidad de $x$.
  5. Interpretacin en regresin mltiple:
    • $\beta_i$: Representa el cambio en $y$ cuando $x_i$ cambia en una unidad, manteniendo las dems variables constantes.

Supuestos del Modelo de Regresin Lineal

Para que los resultados de la regresin sean vlidos, se deben cumplir ciertos supuestos:

  1. Linealidad:
    • La relacin entre $y$ y $x_i$ debe ser lineal. Esto puede verificarse visualizando los residuos vs. los valores predichos.

  1. Independencia de los Errores:

    • Los errores ($\epsilon$) deben ser independientes entre s.

  2. Homoscedasticidad:

    • La varianza de los errores debe ser constante para todos los valores de $x$. Si los residuos muestran un patrn (e.g., cono), hay heteroscedasticidad.

  1. Normalidad de los Errores:
    • Los errores deben seguir una distribucin normal. Se puede verificar con:
      • Grfico Q-Q (Quantile-Quantile).
      • Pruebas de normalidad como Shapiro-Wilk.

  1. No Colinealidad (en regresin mltiple):
    • Las variables independientes no deben estar altamente correlacionadas entre s, ya que esto puede distorsionar los coeficientes ($VIF > 10$ indica colinealidad).

Evaluacin del Modelo

  1. $R^2$: Mide el porcentaje de variabilidad de $y$ explicado por $x$. Un $R^2$ alto indica un buen ajuste, pero no necesariamente un modelo adecuado.

  2. $R^2$ Ajustado: Similar a $R^2$, pero penaliza por el nmero de variables en el modelo, til para regresin mltiple.

  3. Prueba F: Evala si el modelo global es significativo ($H_0: \beta_1 = \beta_2 = \dots = \beta_k = 0$).

  4. Anlisis de Residuos: Residuos (diferencias entre valores observados y predichos) se analizan para detectar violaciones a los supuestos.

Interpretacin y Accin

  • Si los coeficientes son significativos y los supuestos se cumplen, el modelo puede ser usado para prediccin o inferencia.
  • Si los supuestos no se cumplen, considera transformaciones (e.g., logaritmos, polinomios) o mtodos alternativos como regresin robusta o no lineal.

Resumen de ANOVA (Anlisis de Varianza)

El ANOVA (Analysis of Variance) es una tcnica estadstica utilizada para comparar las medias de tres o ms grupos y determinar si existen diferencias significativas entre ellas. ANOVA analiza la variabilidad dentro y entre grupos para evaluar si las diferencias observadas son estadsticamente significativas.

Tipos de ANOVA

  1. ANOVA de una va (One-Way ANOVA):
    • Compara las medias de un factor categrico con varios niveles (grupos).
  2. ANOVA de dos vas (Two-Way ANOVA):
    • Examina dos factores simultneamente, con o sin interaccin entre ellos.
  3. ANOVA con medidas repetidas:
    • Analiza datos en los que las mismas unidades estn medidas en diferentes condiciones o momentos.

Hiptesis

  • Hiptesis nula ($H_0$): Las medias de los grupos son iguales: $\mu_1 = \mu_2 = \dots = \mu_k$.
  • Hiptesis alternativa ($H_a$): Al menos una media es diferente.

Componentes de Variabilidad

El ANOVA divide la variabilidad total ($SST$) en dos componentes principales:

  1. Variabilidad entre grupos ($SSB$):
    • Variabilidad explicada por las diferencias entre las medias de los grupos.
  2. Variabilidad dentro de grupos ($SSW$):
    • Variabilidad debida a las diferencias dentro de cada grupo.

La estadstica $F$ se calcula como: $$ F = \frac{\text{MSB}}{\text{MSW}} $$ Donde:

  • $\text{MSB} = \frac{SSB}{k-1}$ (Media de cuadrados entre grupos).
  • $\text{MSW} = \frac{SSW}{n-k}$ (Media de cuadrados dentro de grupos).

Un $F$ alto sugiere que las diferencias entre grupos no son debidas al azar.

Supuestos de ANOVA

Para que los resultados de ANOVA sean vlidos, se deben cumplir los siguientes supuestos:

  1. Independencia de las Observaciones:

    • Los datos de cada grupo deben ser independientes entre s.

  2. Normalidad:

    • Las distribuciones dentro de cada grupo deben ser aproximadamente normales. Esto se evala mediante:
      • Grficos Q-Q.
      • Pruebas de normalidad (Shapiro-Wilk, Kolmogorov-Smirnov).

  3. Homoscedasticidad (Igualdad de Varianzas):

    • Las varianzas de los grupos deben ser iguales.
    • Se evala con pruebas como Levene o Bartlett.

Pasos para Realizar ANOVA

  1. Plantear la hiptesis nula y alternativa.
  2. Calcular la estadstica $F$:
    • Dividiendo la variabilidad entre grupos ($MSB$) por la variabilidad dentro de grupos ($MSW$).
  3. Evaluar el valor $p$:
    • Si $p < 0.05$, se rechaza $H_0$, indicando diferencias significativas.
  4. Realizar pruebas post hoc (si se rechaza $H_0$):
    • Para identificar qu grupos son diferentes (e.g., Tukey, Bonferroni).

Interpretacin de Resultados

  1. Estadstica $F$:
    • Un $F$ alto sugiere que las diferencias entre grupos son significativas.
  2. Valor $p$:
    • Si $p < 0.05$: Existe evidencia significativa para rechazar $H_0$.
    • Si $p \geq 0.05$: No se rechaza $H_0$; no hay diferencias significativas.
  3. Pruebas post hoc:
    • Identifican especficamente qu pares de grupos difieren.

Aplicaciones de ANOVA

  • Comparacin de tratamientos en estudios experimentales.
  • Evaluacin de diferencias entre grupos en encuestas o estudios de mercado.
  • Pruebas de hiptesis en anlisis usabilidad y diferencias entre grupos de usuarios.

Al cumplir los supuestos y analizar los resultados cuidadosamente, ANOVA es una herramienta poderosa para evaluar diferencias entre grupos.
