Resumen de Regresión Lineal¶

La regresión lineal es una técnica estadística que modela la relación entre una variable dependiente ($y$) y una o más variables independientes ($x$), utilizando una ecuación lineal:

$$ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_k x_k + \epsilon $$

Donde:

  • $\beta_0$ es el intercepto o término independiente.
  • $\beta_1, \beta_2, \dots, \beta_k$ son los coeficientes asociados a las variables independientes.
  • $\epsilon$ es el término de error, que representa la variabilidad no explicada.

Análisis de los Coeficientes¶

Los coeficientes $\beta_i$ indican la magnitud y la dirección del efecto de cada variable independiente sobre la variable dependiente:

  1. Magnitud:
    • Un coeficiente grande implica un cambio significativo en $y$ por cada unidad de cambio en $x_i$, manteniendo las demás variables constantes.
  2. Signo:
    • $\beta_i > 0$: Relación positiva (al aumentar $x_i$, $y$ también aumenta).
    • $\beta_i < 0$: Relación negativa (al aumentar $x_i$, $y$ disminuye).
  3. Significancia:
    • Se evalúa mediante pruebas de hipótesis ($H_0: \beta_i = 0$) usando el valor $p$. Si $p < 0.05$, el coeficiente se considera significativo.
  4. Interpretación en regresión simple:
    • $\beta_1$: Representa el cambio promedio en $y$ por cada unidad de $x$.
  5. Interpretación en regresión múltiple:
    • $\beta_i$: Representa el cambio en $y$ cuando $x_i$ cambia en una unidad, manteniendo las demás variables constantes.

Supuestos del Modelo de Regresión Lineal¶

Para que los resultados de la regresión sean válidos, se deben cumplir ciertos supuestos:

  1. Linealidad:
    • La relación entre $y$ y $x_i$ debe ser lineal. Esto puede verificarse visualizando los residuos vs. los valores predichos.

  1. Independencia de los Errores:

    • Los errores ($\epsilon$) deben ser independientes entre sí.

  2. Homoscedasticidad:

    • La varianza de los errores debe ser constante para todos los valores de $x$. Si los residuos muestran un patrón (e.g., cono), hay heteroscedasticidad.

  1. Normalidad de los Errores:
    • Los errores deben seguir una distribución normal. Se puede verificar con:
      • Gráfico Q-Q (Quantile-Quantile).
      • Pruebas de normalidad como Shapiro-Wilk.

  1. No Colinealidad (en regresión múltiple):
    • Las variables independientes no deben estar altamente correlacionadas entre sí, ya que esto puede distorsionar los coeficientes ($VIF > 10$ indica colinealidad).

Evaluación del Modelo¶

  1. $R^2$: Mide el porcentaje de variabilidad de $y$ explicado por $x$. Un $R^2$ alto indica un buen ajuste, pero no necesariamente un modelo adecuado.

  2. $R^2$ Ajustado: Similar a $R^2$, pero penaliza por el número de variables en el modelo, útil para regresión múltiple.

  3. Prueba F: Evalúa si el modelo global es significativo ($H_0: \beta_1 = \beta_2 = \dots = \beta_k = 0$).

  4. Análisis de Residuos: Residuos (diferencias entre valores observados y predichos) se analizan para detectar violaciones a los supuestos.

Interpretación y Acción¶

  • Si los coeficientes son significativos y los supuestos se cumplen, el modelo puede ser usado para predicción o inferencia.
  • Si los supuestos no se cumplen, considera transformaciones (e.g., logaritmos, polinomios) o métodos alternativos como regresión robusta o no lineal.

Resumen de ANOVA (Análisis de Varianza)¶

El ANOVA (Analysis of Variance) es una técnica estadística utilizada para comparar las medias de tres o más grupos y determinar si existen diferencias significativas entre ellas. ANOVA analiza la variabilidad dentro y entre grupos para evaluar si las diferencias observadas son estadísticamente significativas.

Tipos de ANOVA¶

  1. ANOVA de una vía (One-Way ANOVA):
    • Compara las medias de un factor categórico con varios niveles (grupos).
  2. ANOVA de dos vías (Two-Way ANOVA):
    • Examina dos factores simultáneamente, con o sin interacción entre ellos.
  3. ANOVA con medidas repetidas:
    • Analiza datos en los que las mismas unidades están medidas en diferentes condiciones o momentos.

Hipótesis¶

  • Hipótesis nula ($H_0$): Las medias de los grupos son iguales: $\mu_1 = \mu_2 = \dots = \mu_k$.
  • Hipótesis alternativa ($H_a$): Al menos una media es diferente.

Componentes de Variabilidad¶

El ANOVA divide la variabilidad total ($SST$) en dos componentes principales:

  1. Variabilidad entre grupos ($SSB$):
    • Variabilidad explicada por las diferencias entre las medias de los grupos.
  2. Variabilidad dentro de grupos ($SSW$):
    • Variabilidad debida a las diferencias dentro de cada grupo.

La estadística $F$ se calcula como: $$ F = \frac{\text{MSB}}{\text{MSW}} $$ Donde:

  • $\text{MSB} = \frac{SSB}{k-1}$ (Media de cuadrados entre grupos).
  • $\text{MSW} = \frac{SSW}{n-k}$ (Media de cuadrados dentro de grupos).

Un $F$ alto sugiere que las diferencias entre grupos no son debidas al azar.

Supuestos de ANOVA¶

Para que los resultados de ANOVA sean válidos, se deben cumplir los siguientes supuestos:

  1. Independencia de las Observaciones:

    • Los datos de cada grupo deben ser independientes entre sí.

  2. Normalidad:

    • Las distribuciones dentro de cada grupo deben ser aproximadamente normales. Esto se evalúa mediante:
      • Gráficos Q-Q.
      • Pruebas de normalidad (Shapiro-Wilk, Kolmogorov-Smirnov).

  3. Homoscedasticidad (Igualdad de Varianzas):

    • Las varianzas de los grupos deben ser iguales.
    • Se evalúa con pruebas como Levene o Bartlett.

Pasos para Realizar ANOVA¶

  1. Plantear la hipótesis nula y alternativa.
  2. Calcular la estadística $F$:
    • Dividiendo la variabilidad entre grupos ($MSB$) por la variabilidad dentro de grupos ($MSW$).
  3. Evaluar el valor $p$:
    • Si $p < 0.05$, se rechaza $H_0$, indicando diferencias significativas.
  4. Realizar pruebas post hoc (si se rechaza $H_0$):
    • Para identificar qué grupos son diferentes (e.g., Tukey, Bonferroni).

Interpretación de Resultados¶

  1. Estadística $F$:
    • Un $F$ alto sugiere que las diferencias entre grupos son significativas.
  2. Valor $p$:
    • Si $p < 0.05$: Existe evidencia significativa para rechazar $H_0$.
    • Si $p \geq 0.05$: No se rechaza $H_0$; no hay diferencias significativas.
  3. Pruebas post hoc:
    • Identifican específicamente qué pares de grupos difieren.

Aplicaciones de ANOVA¶

  • Comparación de tratamientos en estudios experimentales.
  • Evaluación de diferencias entre grupos en encuestas o estudios de mercado.
  • Pruebas de hipótesis en análisis usabilidad y diferencias entre grupos de usuarios.

Al cumplir los supuestos y analizar los resultados cuidadosamente, ANOVA es una herramienta poderosa para evaluar diferencias entre grupos.
