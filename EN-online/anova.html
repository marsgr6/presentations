One-way ANOVA

El anlisis de varianza de un solo factor, o ANOVA de un solo factor, es una tcnica estadstica utilizada para determinar si hay diferencias significativas entre las medias de tres o ms grupos independientes. En otras palabras, se utiliza para comparar las medias de dos o ms grupos para determinar si al menos uno de los grupos difiere significativamente de los dems. Esta tcnica es til cuando se tienen mltiples grupos y se desea determinar si al menos uno de ellos se comporta de manera significativamente diferente en relacin con la variable de inters.

El anlisis de varianza de un solo factor se lleva a cabo de acuerdo a los siguientes pasos:

  1. Formulacin de hiptesis:

    • Hiptesis nula (H0): $\mu_1 = \mu_2 = \mu_3 = \dots = \mu_k$, donde $\mu_i$ es la media del i-simo grupo.
    • Hiptesis alternativa (H1): Al menos una de las medias es diferente de las dems.

  2. Recopilacin de datos: Se recopilan los datos $x_{ij}$, donde $i$ denota el grupo y $j$ denota la observacin dentro del grupo.

  3. Verificacin de supuestos: Se verifican los supuestos de normalidad y homogeneidad de varianzas.

  4. Clculo de la suma de cuadrados total (SCT): $$SCT = \sum_{i=1}^{k} \sum_{j=1}^{n_i} (x_{ij} - \bar{x}_{\cdot\cdot})^2$$ Donde $n_i$ es el tamao del grupo $i$ y $\bar{x}_{\cdot\cdot}$ es la media global.

  5. Clculo de la suma de cuadrados entre grupos (SCG): $$SCG = \sum_{i=1}^{k} n_i (\bar{x}_{i\cdot} - \bar{x}_{\cdot\cdot})^2$$ Donde $\bar{x}_{i\cdot}$ es la media del grupo $i$.

  6. Clculo de la suma de cuadrados dentro de grupos (SCW): $$SCW = SCT - SCG$$

  7. Clculo de grados de libertad (GL):

    • Grados de libertad entre grupos: $GL_{\text{entre}} = k - 1$
    • Grados de libertad dentro de grupos: $GL_{\text{dentro}} = N - k$, donde $N$ es el nmero total de observaciones.

  8. Clculo de las medias cuadrticas (MC):

    • Media cuadrtica entre grupos: $MC_{\text{entre}} = \frac{SCG}{GL_{\text{entre}}}$
    • Media cuadrtica dentro de grupos: $MC_{\text{dentro}} = \frac{SCW}{GL_{\text{dentro}}}$

  9. Obtencin de estadsticas de prueba F:

    • Estadstica de prueba F: $F = \frac{MC_{\text{entre}}}{MC_{\text{dentro}}}$

  10. Toma de decisiones: Se compara el valor de $F$ con el valor crtico de la distribucin F para determinar si se rechaza o no $H0$.

  11. Interpretacin de resultados: Si el valor de $F$ es significativo, se concluye que al menos una de las medias es diferente. Se realiza un anlisis post-hoc para identificar cules grupos difieren entre s.

Este proceso proporciona informacin sobre si existen diferencias significativas entre los grupos y cules podran ser esas diferencias.

Anlisis post-hoc

El anlisis post-hoc, tambin conocido como anlisis a posteriori, se utiliza en estadstica despus de realizar pruebas de comparacin mltiple, como el anlisis de varianza (ANOVA) o las pruebas de hiptesis entre varios grupos. El objetivo del anlisis post-hoc es identificar qu grupos especficos son significativamente diferentes entre s despus de que se ha encontrado una diferencia general entre los grupos.

Este anlisis implica realizar una serie de comparaciones entre pares de grupos para determinar si existen diferencias significativas en las medias de las variables de inters. Se utilizan diferentes mtodos estadsticos, como la prueba de Tukey, la prueba de Bonferroni, la prueba de Scheff, entre otras, para realizar estas comparaciones.

El anlisis post-hoc es importante porque ayuda a evitar conclusiones errneas al realizar mltiples comparaciones. Ayuda a identificar qu comparaciones especficas son significativas y cules no lo son, lo que proporciona una comprensin ms precisa de las diferencias entre grupos en un estudio.

Pairwise TukeyHSD

El mtodo pairwise TukeyHSD (Tukey Honest Significant Differences) es una tcnica comnmente utilizada en anlisis post-hoc para comparar todas las combinaciones posibles entre los grupos en un ANOVA (Anlisis de Varianza). Su objetivo es identificar diferencias significativas entre las medias de los grupos despus de haber realizado una prueba global de diferencia entre ellos.

La prueba TukeyHSD compara todas las posibles combinaciones de pares de grupos y calcula intervalos de confianza ajustados para la diferencia entre las medias de cada par. Luego, determina si estas diferencias son estadsticamente significativas en funcin de la variabilidad dentro de los grupos y el tamao total de la muestra.

La ventaja del mtodo TukeyHSD es su capacidad para controlar el error de tipo I cuando se realizan mltiples comparaciones. Al ajustar los intervalos de confianza para cada comparacin, ayuda a mantener un nivel global de significancia, reduciendo as la probabilidad de falsos positivos.

TukeyHSD es una herramienta til para identificar qu combinaciones especficas de grupos difieren significativamente entre s despus de realizar un ANOVA, proporcionando una comprensin ms detallada de las diferencias entre grupos en un estudio.

Supuestos

En un anlisis de varianza (ANOVA), los supuestos fundamentales son:

  1. Independencia de las observaciones: Se asume que las observaciones dentro de cada grupo son independientes entre s y que no hay relacin sistemtica entre las observaciones de un grupo y otro.

  2. Normalidad de los residuos: Se asume que los residuos (diferencias entre los valores observados y los valores predichos) siguen una distribucin normal. Esto significa que los datos en cada grupo siguen una distribucin normal.

  3. Homogeneidad de las varianzas: Se supone que las varianzas de las muestras en todos los grupos son iguales. Esto implica que la variabilidad dentro de cada grupo es similar en todos los grupos.

Estos supuestos son importantes para garantizar la validez de los resultados del ANOVA. Si alguno de estos supuestos no se cumple, los resultados del anlisis pueden ser sesgados o poco confiables. Se pueden realizar pruebas y diagnsticos estadsticos para evaluar si se cumplen estos supuestos antes de interpretar los resultados del ANOVA.

Independencia

Esta suposicin se prueba al disear el estudio. Lo que esto significa es que todos los grupos son mutuamente excluyentes, es decir, un individuo solo puede pertenecer a un grupo. Adems, esto significa que los datos no son medidas repetidas (no se recopilan a lo largo del tiempo). En nuestros ejemplos, esta condicin se cumple.

Normalidad

La suposicin de normalidad se prueba en los residuos del modelo cuando proviene de un marco de ANOVA o regresin. Un mtodo para probar la suposicin de normalidad es la prueba de Shapiro-Wilk. Esto se puede completar utilizando el mtodo shapiro() de scipy.stats.

El test de Shapiro-Wilk es una prueba de normalidad que evala si una muestra proviene de una distribucin normal. La hiptesis nula es que los datos estn distribuidos normalmente. El estadstico de prueba, denotado como $W$, se calcula a partir de las covarianzas entre los valores ordenados y los esperados bajo la hiptesis nula. Si el valor p asociado al estadstico de prueba es menor que el nivel de significancia elegido (generalmente 0.05), se rechaza la hiptesis nula y se concluye que los datos no siguen una distribucin normal. En resumen, el test de Shapiro-Wilk proporciona una medida de qu tan bien se ajustan los datos a una distribucin normal, con valores pequeos de $W$ indicando desviaciones significativas de la normalidad.

El test de Shapiro-Wilk utiliza la siguiente frmula para calcular el estadstico de prueba, denotado como $ W $:

$$ W = \frac{(\sum_{i=1}^{n} a_i x_{(i)})^2}{\sum_{i=1}^{n} (x_i - \bar{x})^2} $$

Donde:

  • $ n $ es el tamao de la muestra.
  • $ x_{(i)} $ es el i-simo valor ms pequeo en la muestra ordenada.
  • $ x_i $ es el i-simo valor de la muestra.
  • $ \bar{x} $ es la media de la muestra.
  • $ a_i $ son los coeficientes especficos para el clculo del estadstico de prueba, que dependen del tamao de la muestra y estn tabulados para diferentes tamaos de muestra.

Una vez calculado $ W $, se compara con un valor crtico de la distribucin de probabilidad bajo la hiptesis nula de normalidad para obtener el valor p asociado. Si el valor p es menor que el nivel de significancia elegido (generalmente 0.05), se rechaza la hiptesis nula y se concluye que los datos no siguen una distribucin normal.

Homogeneidad de las varianzas

El supuesto final es que todos los grupos tienen varianzas iguales. Un mtodo para probar este supuesto es el test de Levene para la homogeneidad de varianzas. Esto se puede realizar utilizando el mtodo levene() de Scipy.stats.

La prueba de Levene se utiliza para evaluar si las varianzas de varios grupos son estadsticamente iguales o diferentes. En otras palabras, se utiliza para probar la homogeneidad de las varianzas entre los grupos. La prueba compara las varianzas de los grupos y determina si existe una diferencia significativa entre ellas. Si el valor p asociado con la prueba de Levene es menor que un umbral predefinido (generalmente 0.05), se rechaza la hiptesis nula de homogeneidad de varianzas, lo que sugiere que al menos una de las varianzas de los grupos es significativamente diferente de las dems. Por otro lado, si el valor p es mayor que el umbral, se acepta la hiptesis nula y se concluye que las varianzas de los grupos son similares.

La prueba de Levene compara la magnitud de las desviaciones de las observaciones dentro de cada grupo respecto a la media del grupo, utilizando una medida de dispersin. La frmula matemtica para el estadstico de Levene es:

$$ W = \frac{(N-k)}{(k-1)} \frac{\sum_{i=1}^{k} N_i (\bar{z}_{i\cdot} - \bar{z}_{\cdot\cdot})^2}{\sum_{i=1}^{k} \sum_{j=1}^{N_i} (z_{ij} - \bar{z}_{i\cdot})^2} $$

Donde:

  • $ N $ es el nmero total de observaciones.
  • $ k $ es el nmero de grupos.
  • $ N_i $ es el nmero de observaciones en el grupo $ i $.
  • $ \bar{z}_{i\cdot} $ es la media de las observaciones en el grupo $ i $.
  • $ \bar{z}_{\cdot\cdot} $ es la media de todas las observaciones.
  • $ z_{ij} $ es la observacin $ j $ en el grupo $ i $.

El estadstico $ W $ sigue una distribucin chi-cuadrado con $ k-1 $ grados de libertad, donde $ k $ es el nmero de grupos.

Kruskal-Wallis

La prueba de Kruskal-Wallis es una prueba no paramtrica utilizada para determinar si hay diferencias significativas entre tres o ms grupos independientes en una variable ordinal o continua. Se utiliza como una alternativa a la prueba de ANOVA cuando los datos no cumplen con los supuestos de normalidad y homogeneidad de varianza necesarios para la prueba de ANOVA.

La prueba de Kruskal-Wallis clasifica los valores de la variable dependiente en cada grupo y luego evala si las medianas de estos rangos son significativamente diferentes entre los grupos. Si el valor p obtenido es menor que un nivel de significancia predefinido (generalmente 0.05), se rechaza la hiptesis nula y se concluye que hay al menos una diferencia significativa entre

La prueba de Kruskal-Wallis utiliza los rangos de las observaciones dentro de cada grupo para determinar si las medianas de estos rangos son iguales o diferentes entre los grupos. Aqu est la descripcin matemtica de la prueba:

  1. Hiptesis nula (H0): Las medianas de los grupos son iguales.
  2. Hiptesis alternativa (H1): Al menos una mediana de grupo es diferente de las dems.

El estadstico de la prueba se calcula de la siguiente manera:

$$ H = \frac{{12}}{{N(N+1)}} \left[ \sum_{i=1}^{k} \frac{{R_i^2}}{{n_i}} \right] - 3(N+1) $$

Donde:

  • $ H $ es el estadstico de la prueba.
  • $ N $ es el nmero total de observaciones en todos los grupos combinados.
  • $ k $ es el nmero de grupos.
  • $ R_i $ es la suma de los rangos de los elementos del grupo $ i $.
  • $ n_i $ es el tamao del grupo $ i $.

El estadstico $ H $ sigue una distribucin chi-cuadrado con $ k - 1 $ grados de libertad. Se compara el valor $ H $ calculado con el valor crtico de la distribucin chi-cuadrado para determinar si se rechaza la hiptesis nula.

El valor $ p $ se calcula a partir del estadstico $ H $ y se compara con el nivel de significancia predefinido para tomar una decisin sobre la hiptesis nula. Si el valor $ p $ es menor que el nivel de significancia, se rechaza la hiptesis nula y se concluye que hay al menos una diferencia significativa entre las medianas de los grupos.

Post Hoc Dunn test

El Dunn test, tambin conocido como prueba de comparaciones mltiples de Dunn, es una prueba no paramtrica utilizada para realizar comparaciones mltiples entre grupos despus de encontrar una diferencia significativa en un anlisis de varianza no paramtrico, como el test de Kruskal-Wallis. Esta prueba ayuda a identificar cules grupos difieren significativamente entre s despus de rechazar la hiptesis nula en el anlisis de Kruskal-Wallis.

Aqu hay una breve explicacin de cmo funciona el Dunn test:

  1. Rangos de observacin: Primero, se asignan rangos a todas las observaciones de los grupos combinados. Esto se hace independientemente de a qu grupo pertenezca cada observacin.

  2. Clculo de las diferencias de rangos: Luego, se calculan las diferencias de rangos entre todas las parejas de observaciones.

  3. Ajuste de rangos: Los rangos se ajustan para tener en cuenta el tamao de las muestras y se suman para cada grupo.

  4. Clculo de las diferencias promedio de rangos: Se calculan las diferencias promedio de rangos entre cada par de grupos.

  5. Correccin de Bonferroni: Finalmente, se aplica una correccin de Bonferroni para ajustar los valores p obtenidos de las comparaciones mltiples.

El Dunn test es una herramienta til para identificar qu grupos difieren significativamente entre s despus de encontrar una diferencia global significativa en un anlisis de varianza no paramtrico. Es especialmente til cuando se tienen ms de dos grupos y se necesita determinar exactamente cules difieren entre s.
